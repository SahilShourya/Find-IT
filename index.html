⚡ Firebase Setup Required

To enable real-time database, authentication, and image uploads, connect your Firebase project:

1
Go to console.firebase.google.com → Create project named smvdu-findit
2
Enable Authentication (Email/Password) + Firestore Database + Storage
3
Click Web App icon → Copy config → Replace firebaseConfig object in this file's script section
4
In Firestore Rules: allow read, write if authenticated. In Storage Rules: allow read, write if authenticated.
